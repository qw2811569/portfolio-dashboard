# 持倉看板（Portfolio Dashboard）

台股持倉追蹤 · 情報脈絡 · 深度研究 Web app。

## 5 分鐘上手

1. `npm install`
2. 複製 `.env.example` 到 `.env.local` 填 token（見 `docs/SERVER_ACCESS_GUIDE.md`）
3. `npm run dev`（vite · 預設 `127.0.0.1:3002`）
4. 開 `http://127.0.0.1:3002`
