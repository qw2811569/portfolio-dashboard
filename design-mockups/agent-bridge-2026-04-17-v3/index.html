v3 vs v2 變化（why 改）

項目原 v2v3（Round 7 ship spec）
裝置定位 desktop only iOS Safari first（iPhone + iPad），桌機兜底
Hero 大標 editorial「你不用看 log...」 刪除（用戶 R4.7/5.5 刪情緒承諾）
Metric labels ACTIVE SESSIONS / TASKS WITH MOTION / SHIPPED RATIO（英） 在跑的 LLM / 工作清單 / 完成率（中）
Action Center 無 top center，白話「現在你最該做的」+ 兩段式確認
Decision queue 無 pull-based list「等你決策 · N 件」（R6.6 砍 push notification，只 pull）
Mac/VM 區分 💻 / 🖥️ emoji icon + 文字 prefix「MAC」/「VM」（R3 + R6 共識：砍 emoji）
今日故事 timeline 無 commit + N 檔 + +X--Y（R5 ship）
Session dial 大中央 crosshair dial 降級（讓位 Action Center），精簡小 indicator
iOS PWA 無 加 manifest，可「加入主畫面」
