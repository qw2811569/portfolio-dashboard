## 🔧 如果無法連接

### 問題 1: 「無法連接伺服器」

**解決方案：**

1. **確認伺服器運行**
   ```bash
   lsof -i :5173
   ```
   應該看到 node 進程

2. **重啟伺服器**
   ```bash
   pkill -f vite
   cd /Users/chenkuichen/APP/test
   npm run dev
   ```

3. **檢查防火牆**
   - macOS: 系統偏好設定 → 安全性與隱私權 → 防火牆
   - 確保允許 Node.js 或瀏覽器通過

4. **嘗試不同的瀏覽器**
   - Safari
   - Chrome
   - Firefox

5. **清除瀏覽器快取**
   - Chrome: Cmd+Shift+Delete
   - Safari: Cmd+Option+E

### 問題 2: 「連接被拒絕」

**解決方案：**

1. **檢查端口是否被佔用**
   ```bash
   lsof -i :5173
   ```

2. **使用不同端口**
   ```bash
   npx vite --port 3000
   ```
   然後訪問 http://127.0.0.1:3000

### 問題 3: 「SSL 錯誤」

**解決方案：**
- 使用 HTTP 而非 HTTPS
- 訪問：http://127.0.0.1:5173（不是 https）

## 📋 測試步驟

1. **打開終端機**
   ```bash
   curl http://127.0.0.1:5173
   ```
   應該看到 HTML 內容

2. **打開瀏覽器**
   - 地址欄輸入：http://127.0.0.1:5173
   - 按 Enter

3. **檢查頁面**
   - 應該看到「持倉看板」應用
   - 按 F12 打開開發者工具
   - 檢查控制台是否有錯誤

## 🐛 常見錯誤

| 錯誤訊息 | 原因 | 解決方案 |
|----------|------|----------|
| 無法連接伺服器 | 伺服器未運行 | 執行 `npm run dev` |
| 連接被拒絕 | 端口被佔用 | `pkill -f vite` 重啟 |
| 404 Not Found | 路由錯誤 | 檢查 URL 是否正確 |
| SSL 錯誤 | 使用 HTTPS | 改用 HTTP |
