## 一句話總結

**這是一個幫你「記帳、追蹤、分析、學習」的台股投資助手。**

不是自動賺錢機器人，而是讓你**投資決策更有系統**的工具。

---

## 一、這個系統在做什麼？

### 1.1 核心功能（4 個字）

| 功能   | 在做什麼                   | 對你的好處             |
| ------ | -------------------------- | ---------------------- |
| **記** | 記錄你的持倉、買賣、目標價 | 不用靠記憶，隨時查得到 |
| **追** | 追蹤月營收、法說會、財報   | 不會錯過重要訊息       |
| **析** | 分析你的持倉表現、風險     | 知道哪裡該調整         |
| **學** | 從成功失敗中累積經驗       | 下次投資更聰明         |

### 1.2 簡單比喻

```
這個系統 = 投資筆記本 + 記事本 + 分析師 + 教練
```

- **筆記本**：幫你記持倉、成本、目標價
- **記事本**：幫你記法說會、財報、月營收時程
- **分析師**：幫你分析持倉風險、部位是否過大
- **教練**：提醒你檢視投資理由，從錯誤中學習

### 4.4 各項功能詳細說明

#### 1️⃣ 持倉頁（最常用）

**你可以做什麼：**

- 查看每檔股票的損益
- 設定目標價、停損價
- 記錄買進理由（thesis）
- 接收風險警告

**操作步驟：**

```
1. 點擊「持倉」分頁
2. 點擊任一檔股票
3. 填寫：
   - 買進理由：為什麼買這檔？
   - 目標價：預期漲到多少？
   - 停損價：跌到多少要跑？
4. 點擊「儲存」
```

#### 2️⃣ 觀察股頁

**你可以做什麼：**

- 加入有興趣的股票
- 設定提醒條件
- 追蹤法說會、財報日程

**操作步驟：**

```
1. 點擊「觀察股」分頁
2. 點擊「新增」
3. 輸入股票代碼或名稱
4. 設定提醒條件（可選）
5. 點擊「儲存」
```

#### 3️⃣ 事件頁

**你可以做什麼：**

- 查看近期法說會、財報公布
- 追蹤事件結果
- 記錄事件後的股價反應

**操作步驟：**

```
1. 點擊「事件」分頁
2. 查看「即將發生」事件
3. 點擊事件可查看详情
4. 事件結束後填寫「實際結果」
```

#### 4️⃣ 分析頁

**你可以做什麼：**

- 收盤後自動分析持倉
- 深度研究單一股票
- 檢視歷史分析記錄

**操作步驟：**

```
1. 點擊「分析」分頁
2. 點擊「開始分析」
3. 等待 AI 分析完成（約 30 秒）
4. 閱讀分析報告與建議
```

#### 5️⃣ 策略大腦（學習功能）

**你可以做什麼：**

- 查看累積的投資規則
- 了解哪些策略有效、哪些無效
- 避免重複犯同樣錯誤

**操作步驟：**

```
1. 點擊「策略大腦」分頁
2. 查看「有效規則」與「無效規則」
3. 閱讀歷史案例
4. 應用到下次投資決策
```

## 六、常見問題（Q&A）

### Q1：我需要懂很多股票知識才能用嗎？

**A：不需要。**

- 新手：可以用基本功能（記帳、追蹤）
- 老手：可以用進階功能（分析、策略大腦）

系統會在你使用过程中提供提示和建議。

### Q2：資料準確嗎？

**A：分層級。**

| 資料類型     | 準確度 | 建議         |
| ------------ | ------ | ------------ |
| 你輸入的持倉 | 100%   | 請仔細輸入   |
| 證交所股價   | 100%   | 可完全信任   |
| 月營收、財報 | 100%   | 可完全信任   |
| 分析師目標價 | 70-80% | 建議交叉驗證 |

### Q3：多久要用一次？

**A：看你的投資風格。**

| 投資風格   | 建議頻率    | 每次時間 |
| ---------- | ----------- | -------- |
| 當沖       | 每日        | 30 分鐘  |
| 短線（週） | 每週 3-4 次 | 15 分鐘  |
| 中線（月） | 每週 1-2 次 | 10 分鐘  |
| 長線（季） | 每月 1-2 次 | 30 分鐘  |

### Q4：我的資料安全嗎？

**A：安全。**

- 所有資料儲存在你的瀏覽器（localStorage）
- 可選雲端備份（Vercel Blob）
- 不會分享給第三方

### Q5：要付費嗎？

**A：目前免費。**

未來可能推出進階功能（如：更多 API 調用、進階分析），但基本功能永遠免費。

### 9.3 隱私與安全

- **資料儲存**：你的瀏覽器（localStorage）
- **雲端備份**：可選（Vercel Blob，加密）
- **API 金鑰**：儲存在後端，前端看不到
- **第三方分享**：不會分享給任何第三方
