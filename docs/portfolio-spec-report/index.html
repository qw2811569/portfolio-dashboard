Contracts

Insider / Hot / Accuracy Gate / News / Daily

這些不是視覺備忘，而是跨頁產品契約。

Insider Holding

7865 = 風險管理語氣

  • `meta.isInsider=true` 後，跳過一般 thesis 與 analyst-target scoring。
  • 只記錄數字、事件、公告與 compliance 提示，不做鼓動性買賣語氣。
  • Trade / Log / Research / PDF 都要繼承這條規則。
Hot Rule

暖色只能是 accent

  • hot 只用在 warning、risk node、underperform gap、single CTA。
  • 不可整頁鋪滿，也不可和舊暖土語彙混打。
  • 主體仍由 bone + iron 組成 calm finance 語氣。
Accuracy Gate

AI 內容先過 5 條檢查

  • 來源可追、日期正確、數字可解釋、語氣不超過證據、低信心要降級。
  • 所有 AI badge pre-display 前必須先過 gate。
  • 幻覺容忍度 = 0；錯一次，信任直接崩。
News Contract

News 不是 Event

  • News 只做 intake / context / source labeling，不在此頁直接生成事件。
  • 需要判斷影響時，handoff 到 Daily，不直接在 News 下結論。
  • `recordType` 分流已決議，不再回頭混用。
Daily Principle

只放 Dashboard

  • R64/R65 定案：Daily Principle Card 只留 Dashboard。
  • 格式：大字 serif 語錄 + 作者 + 今日 context line。
  • Daily 頁維持高密度分析，不再多塞 editorial 卡。
Runtime Contract

Page UI 之前先有閉環

  • 共享 state：portfolio、operatingContext、dossier、newsEvents、tradeLog、analysisHistory、researchHistory。
  • 關鍵 handoff：events 交給 daily、trade 同步到 holdings 與 log、research 匯出 weekly PDF。
  • route shell 只算 migration preview，不是正式產品邊界。
