## 進一步優化嘗試

### 目標

移除 **62 個重複函數**（已在 lib/ 中存在）：
- Holdings: 12 函數 (~190 行)
- Market: 4 函數 (~70 行)
- Datetime: 4 函數 (~80 行)
- Events: 13 函數 (~340 行)
- Brain: 29 函數 (~780 行)
- **總計：** ~1,460 行

### 遇到的挑戰

1. **循環依賴風險**
   - lib/events.js 依賴 App.jsx 的某些函數
   - lib/brain.js 缺少部分 exports

2. **Import 複雜性**
   - 62 個函數需要正確 import
   - 部分函數名稱衝突

3. **測試覆蓋率**
   - 需要完整回歸測試
   - 目前無自動化測試

### 實際測試結果

嘗試刪除重複函數後：
- **成功減少：** ~1,600 行 (從 6,944 → 5,322)
- **Build 錯誤：** 多個 import 錯誤
- **修復時間：** 預計 4-6 小時
