## 服務 Mission 五原則的最強 pattern

### 服務 #1 Zero-Click Awareness（一打開馬上知道狀況）

**Tier S（必抄）**：A · C · E · H · O

- **A 整頁單焦點** + **C 巨型 page title hero** = 「進入即知是哪頁」
- **E Card-stacked spotlight** + **H Split-tile focus state** = 「同階多 entity 立刻知道哪個是焦點」
- **O Embedded ticker chip in news** = 「文章還沒點開就知道相關股當下狀態」

### 服務 #2 漸進式披露（用戶最強調 ⭐️）

**Tier S（必抄）**：L · F · K · M · N · BB

- **L Accordion `..` 收合** = 全研究最直接的漸進披露機制 — Ref 06 Klar 主軸
- **F Time-period segmented** = 維度切換（同資料不同切片）
- **K Stepper form 側邊 index** = 多步表單漸進披露
- **M Horizontal stacked ratings bar** = 一行多比例漸進可視化
- **N Target-range slider** = 一條 bar 解 3 個對照數字
- **BB Heat map** = 從整體 → 局部漸進

### 服務 #3 散戶教學（Beginner-Friendly Guidance）

**Tier S（必抄）**：W · X · AA · O · V

- **W Per-card micro-copy** ⭐️ = 每張卡告訴 user「這數字重要在哪」 — **用戶最該抄的**
- **X Health-score evaluative metaphor** ⭐️ = 用「健康度」「評分」包裝抽象 — 散戶秒懂
- **AA AI insight inline tag** = 「AI 看完幫你整理一句話」
- **O News + ticker chip** = 看新聞前先知道持倉影響
- **V Watch Video / Help 浮動 helper** = 教學不打斷主流程

### 服務 #4 美感（Aesthetic = Retention）

**Tier S（必抄）**：C · D · P · Y · Z

- **C 巨型 page title hero** = 「字級即 hierarchy」
- **D 全站 1 accent CTA** = color discipline 嚴
- **P Sunburst 重量感視覺** = 給數字 mass
- **Y Dark mode + saturate accent** = 質感
- **Z 3D / illustration hero context** = 數字加 emotional context

### 服務 #5 動畫 / 互動（台灣最缺 ⭐️）

**Tier S（必抄）**：F · H · L · Y · P

- **F Time-period segmented 切換** → smooth chart morph
- **H Split-tile focus** → 整塊變色 transition
- **L Accordion** → height auto fade
- **Y Dark mode toggle** → 整頁 fade-cross
- **P Sunburst** → ambient breathing animation
