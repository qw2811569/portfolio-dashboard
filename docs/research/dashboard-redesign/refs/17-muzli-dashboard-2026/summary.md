## 跟 2024 版的 4 個重大趨勢位移（**這是 Round 1 最關鍵發現**）

### Δ1：每 entry 從「by X for Y」一行 → **整段策展分析**

2024 版每個 entry 只有：

```
Project Name by Designer for Studio
> 🔗 [Designer](url)
> 🔗 [Studio](url)
```

2026 版改成完整 H2 標題 + **2-4 句策展段落**，例：

> 「A futuristic workspace designed to optimize the most valuable resource: time. ... features an advanced "Time Breakdown Trend" that categorizes activities into Deep Work, Shallow Work, and Team Meetings, helping users identify their peak focus periods. With integrated AI insights, it tracks event completion rates and cancellation trends, offering a 95% 'Solid Works' focus score to gamify productivity.」

**對本專案啟示（mission #3 · 散戶教學引導）**：muz.li 自己都覺得「光有圖不夠」，要解釋「這個 dashboard 解什麼問題、用哪個機制」。**我們持倉看板的每張卡也應該帶 1 行 micro-copy** — 不只給數字，還告訴 user「為什麼這數字重要 / 這數字現在有沒有問題」。

### Δ2：Domain 從「金融 + Credit Score 主導」→ **更分散 + healthcare/wellness 大幅崛起**

| Domain                            | 2024 版（39 entry） | 2026 版（49 entry） | 變化         |
| --------------------------------- | ------------------- | ------------------- | ------------ |
| Fintech / 金融                    | 12 (31%)            | ~10 (20%)           | **降**       |
| Credit Score                      | 5 (13%)             | 0                   | **整類沒了** |
| Healthcare / Wellness             | 1 (3%)              | 5 (10%)             | **升 3x**    |
| SaaS Analytics                    | 2 (5%)              | 8 (16%)             | **升 3x**    |
| AI / Smart Workspace              | 0                   | 4 (8%)              | **新類**     |
| E-commerce                        | 1 (3%)              | 3 (6%)              | 升           |
| Real Estate / Climate / Time-mgmt | 0                   | 4 (8%)              | **新類**     |

**對本專案啟示（mission #4 · 美感 / mission #5 · 互動）**：2024 是金融主導，2026 是「**所有 domain 都向 SaaS Analytics 靠攏**」。這正面：我們做投資 dashboard 可同時 borrow healthcare/wellness 的「**柔軟、計時、健康評分**」隱喻 — 把「持倉健康度」「曝險警示」用類似 health score 的視覺語言呈現。

### Δ3：黑底 / Dark mode 比例顯著增加

2024 只有 Ref 13 Ronas IT 是 dark mode；2026 有 **img-10（CryptoZen / dark）· img-13（ZenWallet）· img-15（Fintech Wallet）· img-37（Analytics Dashboard – Dark Mode UI）** 4+ 個明確 dark-mode 主視覺。

**對本專案啟示**：投資人**凌晨看盤** = dark mode 剛需。Ref 16 Startify 的「黑底 panel = realtime」可進化成「整 page dark mode toggle + realtime 用更 saturate accent」。

### Δ4：Hero 數字字級更陡 + 3D / 圖像化元素增加

img-29（Hynex Healthcare）右側用 **3D 立體 stomach illustration** 當 hero 視覺；img-42（Noteflow）用 **戶外草地 + 天空背景**包覆 dashboard。2024 版多半是純白 / 純 sage 平面 — 2026 開始**用視覺隱喻給數字 emotional context**。

**對本專案啟示（mission #4 美感）**：純數字 + 純圖表的時代正在過。可考慮：

- 「總損益」加一個微妙視覺意象（如綠色山脊 / 紅色山谷）
- 「持倉健康度」配象徵性插畫（如水晶 / 心電圖）
- 但**插畫不能搶數字焦點**，必須降到第二層
