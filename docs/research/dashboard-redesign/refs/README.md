## 描述檔模板

```markdown
# Ref NN · <slug>

- **來源**：URL / IG handle / Dribbble / 書 / 截圖
- **取得日期**：YYYY-MM-DD
- **由誰提出**：用戶 / Codex / Gemini
- **檔案**：`./NN-<slug>/01.png`, `02.png`, ...

## 為什麼存這份

（用戶想參考的點 — 1-2 句）

## 視覺 / 互動觀察（Claude 看完寫）

- 配色 / 字型 / 留白
- 主互動模式（focus / drill / scroll / swipe）
- 資料密度（高 / 中 / 低）
- 動效角色

## 可借鑒到本專案的點

- ...

## 不適用本專案的點

- ...
```
