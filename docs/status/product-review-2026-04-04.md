# 產品經理 UX 審查 — 2026-04-04

## 核心價值主張

> 幫台股散戶把每天看盤、做筆記、追事件、檢討操作，從散落在 Line 群和腦袋裡，變成一個有 AI 幫你整理和提醒的系統。

## 快贏改進（已實施 / 進行中）

1. ✅ 持倉頁加「AI 今日快評」摘要卡片
2. 🔄 持倉頁加「今日損益」（agent 進行中）
3. 🔄 研究頁術語白話化（agent 進行中）
4. 🔄 備忘錄可跳過（agent 進行中）

## UX 痛點

1. 八個 Tab 太多，散戶不知道從哪開始 → 考慮收成 4 個主 Tab
2. 研究頁術語太硬（Karpathy、dossier、Gate）→ 改中文白話
3. 交易備忘錄強制填寫壓力大 → 加跳過按鈕

## 缺少功能（未來規劃）

1. 即時報價 / K 線迷你圖（sparkline）
2. 事件通知 / 推播機制
3. 績效追蹤（投組 vs 大盤比較）

## 產品定位

- vs 三竹/XQ：他們是看盤工具，我們是決策工具
- vs TradingView：他們不懂台股生態，我們懂法說會、月營收節奏
- 差異化：唯一串起 OCR 記帳 + AI 分析 + 事件行事曆 + 策略復盤的工具
